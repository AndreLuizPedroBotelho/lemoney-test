### Clone

---

```sh
$ git clone https://github.com/AndreLuizPedroBotelho/lemoney-test.git
```

### Installation

---

```sh
$ cd lemoney-test

$ npm install
```

### Configure

---

Configure env file in backend folders using env.example as a base

### Usage

---

Start projects

```sh
$ npm start
```

---

Test projects

```sh
$ cd backend && npm run test
```

### Tech Stack
---

- node
- lerna
- jest

### Required

---

- node >= 12.19.0
